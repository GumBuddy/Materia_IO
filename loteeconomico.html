Modelo de Lote Económico

Definición



El Modelo de Lote Económico ha sido el modelo de inventario con mayor uso.

Su popularidad se debe a una combinación de simplicidad y de amplia aplicabilidad.

Esta diseñado para el tipo de situación donde el producto debe retirarse del inventario, en esencial, a una tasa constante.
El lote económico tiene como propósito elegir la cantidad a ordenar que sea más económica.

Supuestos del modelo:

  1. La demanda es conocida y ocurre a una tasa constante
  2. Los productos tienen duración suficiente
  3. Se utiliza un sistema de monitoreo continuo
  4. Todos los costos permanecen constantes en el tiempo
  5. La cantidad a ordenar para reabastecer el inventario llega toda a la vez cuando se desea
  6. No se permiten los faltantes planeados
